# 原文 05：采购关系与边界测试

来源说明：

- 主题：采购关系、组织冷感、权力边界
- 用途：支撑“合作本质是采购关系”与“甲方会测试边界”

原文一：

```text
所以，在我还不是老板的年代里，我就明白一件事，公司从来不是你可以依赖的对象。

公司和你之间，只是一种采购关系，就像你是卖面粉的，对方是卖包子的，人家在你这里进货，就这点关系。

对方今天可以在你这里进货，明天就可以不在你这里进货。

它不欠你一份工作。
```

原文二：

```text
就是我前面说的，反复测试他权力的边界，这一点，谁都逃不开，这就是最后的看山还是山。

看富老头还是富老头。

我年轻时跟过的，我们私底下就在调侃，说他完全可以有500人的歌舞团，为什么非要折腾我们这些人，害的我们成天加班。

他最喜欢的事情就是折腾，把一堆500强供应商弄到一起，天花乱坠，胡搞瞎搞。

这就是我们的玩笑话，为什么有的富老头就是喜欢挖运河，就是喜欢三征高丽，有这功夫有这资源，你找500个美女，多省钱，多开心。

我们当年跟的那位，就这样，他就喜欢折腾，我们一天到晚跟在身边，很清楚，他其实吃的也很简单，用的也很简单，生活更加简单。

然后喜欢干嘛呢？喜欢折腾，搞得整个行业大地震，还听不进去劝谏。

其实你回头想想看，他为什么做这些？还是在反复试探边界。
```
